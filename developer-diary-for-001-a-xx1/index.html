Description

This is a final year project for diploma. The purpose of developing this software is to understand how to connect frontend to backend using RESTful API.

Technology

  • Flutter
  • Flask
  • MongoDB

Day 1

🔷 Goal 🔷

  • Design login screen
  • Working login form

Day 2

🔷 Goal 🔷

  • Redesign the login form text field
  • Add snackbar to show the login error

🔴 Problem 🔴

  • Encounter a problem: Use card to materialize the TextFormField will malform its hint and helper text
  • Solution: Redesign the field with InputDecorationTheme and use OutlineInputBorder

☑️ Achievement ☑️


Day 3

🔷 Goal 🔷
Design password reset request form
Add snackbar to notify user whether success or fail
Use jwt to generate link and email it to user
Design simple password reset form in web
Design simple web pages to tell user if whether success or fail
